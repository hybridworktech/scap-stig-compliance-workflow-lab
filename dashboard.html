Employer Value Proposition

I built this privacy-safe SCAP/STIG compliance automation lab to show how I approach infrastructure work: automate the repeatable parts, protect sensitive data, and turn raw technical findings into clear engineering action. The project uses PowerShell, synthetic benchmark-style scan data, Windows read-only validation checks, remediation tracking, before/after reporting, and generated runbooks. It is especially relevant to DevOps, cloud/platform engineering, security automation, GRC, government contracting, and eDiscovery/legal-tech infrastructure roles.
